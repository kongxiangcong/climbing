# Climbing — 产品需求文档

## 产品定位

**面向个人投资者的 A 股研究与持仓跟踪工具**。核心不是替用户做交易决策，而是把“研究结论、证据、计划、偏离、复盘”组织成可追溯的工作流。

## 边界声明

- ✅ 做研究、持仓跟踪、计划管理、报告生成
- ❌ 不做交易执行、不做自动下单、不做盘中微观结构博弈
- ❌ 不做面向第三方的个性化投顾服务

## 核心用户

单用户（系统所有者本人）。设计原则：**高解释性、高可追溯性**。

## 四个主模块

### 1. 个股全景分析与估值报告

输入股票代码，输出：
- 公司档案（主营业务、分部、行业位置、股东结构）
- 财务质量判断
- 经营周期位置
- 估值区间
- 技术位置
- 催化剂与风险点
- 当前操作建议

所有结论必须包含：**假设、证据、失效条件**。

### 2. 持仓管理

支持手动导入持仓 CSV 与交易流水 CSV。自动计算：
- 最新市值、单票浮盈亏、组合浮盈亏
- 已实现收益、总收益率
- 仓位结构、行业暴露、集中度
- 回撤、波动率、Beta、相关性、VaR
- 资金加权收益 + 时间加权收益

### 3. 交易计划创建与跟踪

交易计划是结构化对象，字段包括：
- 计划名称、标的、时间窗口、方向
- 研究结论版本、入场逻辑
- 目标价区间、分批策略、仓位上限、风险预算
- 失效条件、替代情景、触发器、复盘频率

计划状态机：
`草稿 → 激活 → 部分触发 → 完全触发 → 假设被破坏 → 关闭 → 复盘完成`

当行情偏离计划时，系统弹出证据面板，由用户手动确认并留痕。

### 4. 市场与宏观月报

四栏因子面板：
- **增长**：工业、消费、投资、出口
- **通胀**：CPI、PPI
- **流动性**：M2、社融、利率环境
- **市场结构**：指数点位、成交额、估值、融资融券、行业轮动、市场宽度

输出标签：`过热`、`中性`、`偏冷`，附带证据链。

## 数据蓝图

### 四层数据

1. **官方披露层**：财报、公告、股东变动、定期报告、临时公告
2. **结构化市场数据层**：日线行情、财务三表、业务分部、盈利预测、股东信息、财务指标
3. **宏观与市场情绪层**：GDP、CPI/PPI、M2、社融、利率、指数估值
4. **辅助情报层**：卖方研报、新闻舆情、学术、企业工商

### 更新节奏

- **日更**：收盘后刷新价格、公告、持仓摘要、计划偏离
- **事件驱动**：新公告、财报、股东变动、异常价格行为时重算
- **月更**：宏观和市场状态报告

## 分析引擎

三个评分器 + 一个状态机：

1. **个股研究评分器**：满分 100，分基本面质量（30）、趋势与预期差（20）、估值（20）、技术面（15）、风险暴露（15）
2. **市场温度评分器**：六维度合成温度分数
3. **计划偏离评分器**：检查价格触发、趋势逆转、财报miss、公告改变逻辑、仓位越界
4. **研究对象生命周期状态机**：候选 → 研究中 → 已立计划 → 持仓中 → 计划失效 → 观察修复 → 归档

## 数据流

```
采集 → 标准化 → 版本化 → 特征计算 → 规则引擎 → 报告生成 → 人工修订 → 快照归档
```

## 交付路线图

| 阶段 | 目标 | 优先级 |
|------|------|--------|
| 本地研究 MVP | 个股分析、持仓导入、收盘更新、交易计划、告警中心、报告快照 | P0 |
| 研究增强版 | 公告摘要、财报变化解释、估值情景、组合风险仪表盘、计划复盘页 | P1 |
| 市场月报版 | 宏观月报、市场温度、行业轮动面板、宽基指数比较 | P1 |
| 外部数据增强版 | 更稳定研报接入、可比公司映射、企业关系图谱、可选 ESG/专利 | P2 |
| 实时交易化扩展 | 暂不建议 | 暂缓 |

## 验收标准

- 收盘后一键更新成功率高且可重跑
- 任一持仓的收益、累计收益和交易流水可互相校验
- 任一交易计划可追溯到对应研究报告版本
- 任一告警可展开到具体触发条件和证据
- 月报在主要宏观数据发布后一个工作日内自动出稿
- 受限制数据源不得作为权威口径再分发
